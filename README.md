# Backend Ledger Transaction System

A backend system designed to handle secure financial transactions, account management, and accurate balance tracking using RESTful APIs. Built with Node.js and Express, focusing on data consistency and scalable architecture.

# Features
💳 Create and manage user accounts <br>
💰 Perform credit and debit transactions<br>
📊 Real-time balance tracking<br>
🔐 Secure API design<br>
🔁 Idempotent transactions (prevents duplicate payments)<br>
📡 RESTful API architecture<br>
🗄️ Database integration for persistent storage<br>

# Tech Stack
Node.js<br>
Express.js<br>
MongoDB<br>
Mongoose<br>
REST API<br>
